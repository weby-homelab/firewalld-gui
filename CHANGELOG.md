# Changelog / Історія змін

## [1.6.1] - 2026-04-05
- **Security:** Виправлено LFI (Path Traversal) вразливість у REST API.
- **Backend:** Оптимізація роботи з firewalld-cmd (додано кешування станів).

## [1.6.0] - 2026-03-20
- **Features:** Повна підтримка Rich Rules та ICMP-блокування через GUI.
- **Monitoring:** Інтеграція Geo-IP логів для виявлення підозрілих дропів.

## [1.0.0] - 2025-10-01
- Початковий реліз системи керування Firewalld через веб-інтерфейс.
